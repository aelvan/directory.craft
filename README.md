# Analytics *for Craft*

This plugin helps you print Google Analytics tracking code.

## Installation

Put the **analytics/** folder inside **craft/plugins/**.

## Example

Put this code *"after the opening `<body>` tag"* as they say :

    {{craft.analytics.code("UA-XXXXXXX-X")}}
